  ________
 /CONTROLS\
 ===============================================
| down                      = 	down arrow key  |
| left                      = 	left arrow key  |
| right                     = 	right arrow key |
| rotate clockwise          = 	shift           |
| rotate counter-clockwise  = 	control         |
| hard drop                 = 	spacebar        |
| pause                     = 	escape          |
| hold			    =   up arrow key    |
 ===============================================

You can change the key configuration from the options menu.
